# EasyPDF
===

> Diclaimer: Work in progress

EasyPDF is a library for generating pdf
documents from simple xml templates

Template can be
```
<xml>
  <page>
    <text>Hello World</text>
  </page>
</xml>
```

## Goals
===
- Generating files from a simple template
- Reading and editing PDF files
